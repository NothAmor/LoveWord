情言 API 使用说明

1、说明

情言(tddiao.online)创立于2018年，目前网站主要提供一句话服务。

创始人只是很喜欢收集情话，然而自己收集了又没有人可以说，那就开个站放着，也许有有缘人能看见，或者也能让他人收益

简单来说，情言指的就是一句情话


5、使用

https://www.tddiao.online/word（输出纯文字）

https://www.tddiao.online/word/?format=js&charset=utf-8 （进阶版）

网页使用示例:

<!-- 新 API 方法， 十分简洁 -->
<script type="text/javascript" src="https://www.tddiao.online/word/?format=js&charset=utf-8" ></script>
